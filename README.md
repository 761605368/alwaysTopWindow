# utools-vue3-markdown-editor
基于 vue3 + vite + elementui + markd + keyboardjs + highlight.js 实现的清爽 Markdown 编辑器

![screen](./screen.gif)

## 功能

- 支持复制文件触发编辑
- `Ctrl + S` 即时保存，`Ctrl + Shift + S` 另存为
- 支持代码高亮，即时渲染 Github 样式的预览
